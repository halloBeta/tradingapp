# Trading App
- Sample Trading Application that uses Java Spring boot and JMS(ActiveMQ broker)
- In this application user submitted trade is enriched by calling Alpha Vantage API

## Local Setup on windows
- ./mvnw clean install/compile
- ./mvnw spring-boot:run
- Note: to clean up the workspace, do ctrl + shift + p, then on windows palatte run java: clean workspace
